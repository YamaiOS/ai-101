Who this is for

Built for practitioners,
not beginners

For professionals in environments where a wrong AI output has downstream consequences — misclassified exposures, undocumented exceptions, incorrectly sequenced approvals. The quality ceiling here is set by domain knowledge, not prompt technique.

This workshop is for you if…

  • You work in a regulated or consequence-bearing domain
  • Your outputs carry institutional weight
  • You've used AI tools but need reliability, not plausibility
  • Your workflows need to survive audit and scrutiny
  • You're moving from personal experiment to institutional practice
  • You need to present AI architecture to governance stakeholders

This workshop is not for you if…

  • You're a developer building portfolio projects
  • You're a hobbyist exploring AI tools
  • You want a step-by-step Claude Code how-to
  • You're looking for a prompt engineering course
  • You want to learn AI from scratch

Environment setup

You must arrive with a working environment. Two paths are supported. Both end with the same verification step — your seat time depends on it.

  • Node.js version 18 or above
  • Claude Code installed globally via npm: npm install -g @anthropic-ai/claude-code
  • Visual Studio Code
  • Git configured with your name and email
  • A GitHub account
  • A Claude account — your Pro subscription is included and will be activated at the workshop
  • A GitHub account and a modern browser — that's all you need
  • The facilitator provides a template repository to launch from
  • No local installation required — designed for managed corporate devices

Verification (both paths): Open your terminal, type claude, authenticate with your credentials, and ask the model to create a file called ready.txt containing the text "I am set up." If the file appears in your project directory, your environment is confirmed.
Setup Clinic — An online setup clinic runs the evening before the workshop. Details are in your registration confirmation. If your environment is not verified before the day, attending the clinic is strongly recommended.
